资源类型
========

资源是对云平台中用户可以创建、修改、删除、查询的配置、运行时实例等数据的抽象，是云平台所管理的数据资产，因此也就是操作审计的保护对象。
经过对nova、glance、neutron、cinder服务的分析，操作审计将资源分类整理，并给每一类资源分配了唯一标识，用于从资源视角进行查询和审计。

资源的唯一标识是一个包含父子关系的多段标识符，用“."进行分隔。下表是目前支持的资源唯一标识及名称的列表：
